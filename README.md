## 📌 프로젝트 소개  
우주를 배경으로 한 비행 Shooter 게임입니다.

<br></br>
## 🎮 게임 규칙  
1. **화면 위쪽**에서 적들이 생성됩니다.  
2. 적들을 격추시키면 점수 획득!  
3. 적과 충돌 시 Game over  

<br></br>
## 🛠️ 사용 기술 스택  
- **Unreal Engine5**: 게임 개발 및 UI 구성  
- **C++**: 게임 로직 구현  
- **GitHub Actions**: 코드 버전 관리
